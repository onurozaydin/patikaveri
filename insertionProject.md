# Insertion Sort türüne göre aşamaları 

- [22,27,16,2,18,6] ==> n
- [2,27,16,22,18,6] ==> n - 1
- [2,6,16,22,18,27] ==> n - 2
- [2,6,16,18,22,27] ==> 1

# Big-O gösterimi ==> O(n^2)


# Time Complexity

- Best Case ==> 1
- Average Case ==> O(n)
- Worst Case ==> O(n)
